    적립금 사용 설정

    사용 조건

    전체 상품 구매금액이 원 이상일 때 적립금 사용가능

    전체 상품 구매금액 = 판매가격(할인가)+옵션, 프로모션코드, 쿠폰 및 배송비는 제외

    보유 적립금 원 부터 주문 시 사용가능

    사용 제한

    1회 주문시 최대 사용 가능 적립금 원 0원 입력시, 무제한

    주문 결제 중 화면 이탈로 결제가 실패된 경우, 사용한 적립금은 자동 복구됩니다.(결제 실패 1시간 후 자동복구)
